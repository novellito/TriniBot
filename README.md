# triniBot

This is my first bot made using the [Twitter api](https://dev.twitter.com/rest/public).

## Technologies used to make this
* Node.js
* Twitter API
* Express.js
* Express-handlebars
* Jquery
* HTML/CSS/Bootstrap 3

## What can it do?

* Generate Tweets
* Follow Users
* Retweet
* Like posts

## Misc.

bot2.js contains a more basic CLI version of the bot.
